Follow up questions based on this solution.
What is the document?
The Document is what JS uses to interact with the html. In this instance we are searching through the document for an HTML element by a specific id and saving it into a variable.
What is the difference between a class and an id?
We use an id on an html element to give it a unique property. A class is given to a several html elements to that they may share a property.
What is the difference between innerHTML and innerText?
.innerHTML is a method that changes the html of the html element that it is being used on. .inntext is different in the sense that it will not replace the html, but instead will just replace the text of the element with the given data.
What is the difference between Javascript and Jquery?
Javascript is a programming language while Jquery is a javascript library. Basically a collection of javascript to speed along development.
Besides getElementById() how else can we select elements from the HTML?
There are several ways. We can use getElementById() and querySelector() for single elements and getElementsByClassName(), getElementsByTagName(), querySelectorAll() for multiple.

Your Code is using ES6 Syntax with arrow functions. Why ES6 vs ES5 variable declaration?
'let' and 'const' variable declaration have some difference from 'var'. 'let' is function scope specific and 'const' creates a constant variable meaning its value can not be changed. This allows us to have more control over our data types.
